# Como rodar os projetos

## Front

Instale as dependências rodando:

```shell
npm i
```

E então rode o projeto:

```shell
npm run dev
```

## Back

Para rodar o projeto é necessário ter o mongodb instalado. Caso seja necessário altere as configurações do banco no `application.properties`

Para rodar o projeto:

```shell
mvn spring-boot:run -X
```
